### Debian

```shell
curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/StepBroBD/Automations/master/debian.sh | bash
```

### macOS

```shell
curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/StepBroBD/Automations/master/macos.sh | zsh
```
